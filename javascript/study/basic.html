기본 규칙

자바스크립트의 기초적인 특징과 기본 문법에 대해서 잘 숙지하고 공부를 시작하기

자바스크립트란 무엇인가요?

JavaScript는 웹 페이지에서 복잡한 기능을 구현할 수 있도록 하는 스크립트 프로그래밍 언어입니다.
HTML이 정적인 레이아웃을 표현한다면, 자바스크립트는 동적인 기능을 표현해줍니다.

자바스크립트는 1995년 넷스케이프 커뮤니케이션의 브렌던 아이크가 개발하였으며
Netscape Navigsator2.0에 구현되었습니다.
1996년에는 마이크로소프트사의 Internet Explorer3.0 탑재되었으며
java애플릿으로 구현된 웹 브라우저의 지지부진한 보급과 맞물려 빠르게 보급되었습니다.
1997년부터 ESMAScript를 따르는 표준화 작업을 통해 현재는 모든 브라우저에서 지원됩니다.

자바스크립트 기본 규칙

01. 세미콜론 (;)

하나의 실행문이 끝나면, 실행문 마지막에 세미콜론(;)을 붙여 준다.
세미콜론(;)을 붙여 주지 않아도 상관은 없지만, 붙여주는 습관을 들이도록 해보자.

var num = 5 ;
document.write(num) ;

02. 대소문자

자바스크립트 명령어는 대소문자를 구별합니다.
(소문자)console와 (대문자)Console은 각각 다른 문자열로 처리됩니다.

03. 들여쓰기

자바스크립트는 가독성을 위하여 들여쓰기를 사용합니다.
tap또는 space 2번

05. 주석

자바스크립트 주석은 일반적으로 특정 코드에 대한 설명을 입력하거나, 특정한 부분의 코드가 실행되지 않도록 사용한다.
주석 표시는 프로그램에 영향을 미치지 않습니다.

출력하는 방법

자바스크립트에서는 출력할 수 있는 방법이 여러가지 있다.

분류 메서드
입력 prompt() : 화면에 프롬프트를 띄워 사용자가 입력할 수 있는 상자를 만들어낸다.
출력 document.write() : 대부분 출력 테스트를 위해 사용되는 방법
alert() : 웹 화면에 확인 버튼이 있는 경고 대화상자를 띄운다.
console.log() : 웹 브라우저 검사 항목에 있는 콘솔메뉴에 출력을 한다.
확인 confirm() : 확인과 취소 버튼이 있는 대화 상자를 띄운다.

자바스크립트 사용밥법

자바스크립트의 사용방법은
1. 외부파일
2. 스크립트 태그
3. 태그에 직접 기술하는 방법 등이 있다. 

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>Javasript</title>
    <!-- 1. 외부 파일로 로드하는 방법 -->
    <script type="text/Javasript" src="js/script.js"></script>

    <!-- 2. script 태그 사이에 기술하는 방법 -->
    <script>
        document.write("Javasript");
    </script>
</head>
<body>

    <!-- 3. 태그에 직접 기술하는 방법 -->
    <a href="#" onclick="alert('Javasript')"></a>
</body>
</html>
