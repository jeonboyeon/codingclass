조건문

조건문은 조건식이 참인 경우 참의 결과를 실행하고,
거짓인 경우 거짓 조건의 결과를 나타내는 함수이다.

첫번 째 : if 문

if 안에 조건식이 참인 경우 해당하는 참의 결과를 실행한다.
조건문에 해당하지 않고 거짓이면 esle 안에 있는 코드 블록이 실행됩니다.

if ( 조건식 ){
    실행문 ;
} 

var age = 10;                  //   age(나이) 는 10 이다.
  var admissionFee = " ";  //   (입장료)결과 실행 칸
  if (age < 13) {               //   age = 10 이다. 13보다 적으니 참
      admissionFee = "2000원";      //  << 참일 경우 여기의 값을 실행한다.
  }

  console.log(admissionFee);
결과확인
2000원

두번 째 : 중첩 if 문

if문 안에 또 다른 if문을 중첩해서 넣는 형식이다.
중첩 개수에는 제한이 없지만 else 사용 시 {} 사용, 들여쓰기에 주의 해야 한다.

if (조건식1){
    if (조건식2){
        실행문;
    }
} 

var html = Number(prompt('html 점수입력',''));   //  html 점수입력을 할 수 있는 프롬프트 실행
var css = Number(prompt('css 점수입력',''));   //  css 점수입력을 할 수 있는 프롬프트 실행
var javascript = Number(prompt('javascript 점수입력',''));  //  javascript 점수입력을 할 수 있는 프롬프트 실행
var avg = (html + css + javascript) / 3;    //  평균 = html + css + javascript /(나누기) 3
var comment;
if (avg >= 60) {
    if (html < 40) {
        comment = 'html' + '40점 미만 불합격';
    } else if (css < 40) {
        comment = 'css' + '40점 미만 불합격';
    } else if (javascript < 40) {
        comment = 'javascript' + '40점 미만 불합격';
    } else {
        comment = '평균' + avg + '점 합격';
    }
} else {
    comment = '평균' + avg + '점 불합격';
}

document.write(comment);
결과확인
html, css, javascript 모두 더한 점수에서 3을 나눴을 때
평균 60 점 이상이면 합격
평균 60 점 미만이면 불합격
