상수, 배열, 객체, for문, forEach문, addEventListener 를 사용하여 퀴즈 문제 만들기

querySelectorAll을 이용해 class값 요소를 찾아준다.

앞 전 문제들과 달리 문제가 4개 이므로 querySelector이 아닌
querySelectorAll을 이용해 여러 요소를 한 번에 가져온다. 

//선택자
const quizType = document.querySelectorAll(".quiz__type");    // 퀴즈 종류
const quizNumber = document.querySelectorAll(".quiz__question .number")    // 퀴즈 번호
const quizAsk = document.querySelectorAll(".quiz__question .ask")   // 퀴즈 질문
const quizConfirm = document.querySelectorAll(".quiz__answer .confirm")    // 정답 확인 버튼
const quizResult = document.querySelectorAll(".quiz__answer .result")    // 정답 결과
const quizInput = document.querySelectorAll(".quiz__answer .input")    // 사용자 정답   
const quizView = document.querySelectorAll(".quiz__view")   // 강아지 애니메이션

상수의 상수값을 정하고 document.querySelectorAll을 이용해 각각 class값을 모두 찾아준다.

for문 혹은 forEach문을 이용해 값을 넣어준다.

하드 방식 < for문 < forEach문 순서로 간편해지니 forEach문을 습관화 해보자.


for문 사용

for ( i=0; i<quizInfo.length; i++ ){
    quizType[i].textContent = quizInfo[i].answerType;
    uizNumber[i].textContent = quizInfo[i].answerNum;
    quizAsk[i].textContent = quizInfo[i].answerAsk;
    quizResult[i].textContent = quizInfo[i].answerResult;
}

quizType 쪽만 풀어서 설명 해보자면
quizInfo[i].answerType;
"quizInfo[i]" 값의 "answerType" 을
quizType[i].textContent
"quizType[i]" 값에 "textContent" 텍스트로 대입한다.


forEach문 사용

quizInfo.forEach(( e, i ) => {  // function을 없애고 화살표 함수 사용
    quizType[i].textContent = quizInfo[i].answerType;
    quizNumber[i].textContent = quizInfo[i].answerNum;
    quizAsk[i].textContent = quizInfo[i].answerAsk;
    quizResult[i].textContent = quizInfo[i].answerResult;
});

하드 방식, for문 혹은 forEach문을 이용해 정답 칸을 숨겨준다.

자바스크립트 스타일 적용 방식으로 정답확인 버튼 아래 정답 칸을 숨겨보자.


하드 방식

quizResult[0].style.display = "none";
quizResult[1].style.display = "none";
quizResult[2].style.display = "none";
quizResult[3].style.display = "none";

for문 사용

for (let i=0; i<quizResult.length; i++){
    quizResult[i].style.display = "none";
}

(let i=0; i<quizResult.length; i++)
i=0 이고 i는 quizResult.length 개수만큼 i++ 1식 커진다.
그렇게 되면 quizResult[i]의 i 값은 quizResult.length의 값이 4니까
0부터 3까지 (4개니까 4번 반복) 적용이 되어 4개의 문제 모두 답을 숨길 수 있는 것.


forEaxh문 사용

quizInfo.forEach (function(e, i){
    quizResult[i].style.display = "none";
});

정답 확인 버튼을 누를 때 발생한는 이벤트들을 설정한다.

addEventListener 를 이용해 이벤트 발생을 시켜보자.


forEach, addEventListener 사용

quizConfirm.forEach((btn, num) => {  // 버튼 4개가 있다는걸 알려주기위해서 forEach문을 통해 btn 개수를 알려주고 그 다음 클릭 이벤트를 실행
    btn.addEventListener("click", ()=> {

        //사용자 정답
        const userWord = quizInput[num].value.toLowerCase().trim();

        //사용자 정답 == 문제 정답
        if(userWord == quizInfo[num].answerResult){
            //정답
            // alert("정답")
            quizView[num].classList.add("like");
            quizConfirm[num].style.display = "none";
            quizResult[num].style.display = "block";
            quizInput[num].style.display = "none";
        } else {
            //오답
            // alert("오답")
            quizView[num].classList.add("dislike");
            quizConfirm[num].style.display = "none";
            quizResult[num].style.display = "block";
            quizInput[num].style.display = "none";
        }
    })
});

quizConfirm.forEach((btn, num) : btn, num의 모든 요소를 호출한다.
btn.addEventListener("click", ()=> : 호출 하여 찾은 버튼들에게 클릭할 시 발생할 이벤트를 정한다.
if(userWord == quizInfo[num].answerResult) : userWord와 quizInfo[num].answerResult 의 답이 같을 시
실행할 것을 정해준다.

답이 같을 시
quizView[num].classList.add("like") : quizView[num] 각각에 like 클래스를 적용
quizConfirm[num].style.display = "none" : quizConfirm[num] 각각을 화면에서 없앤다.
quizResult[num].style.display = "block" : quizResult[num] 각각을 화면에 다시 나타나게 해준다.
quizInput[num].style.display = "none" : quizInput[num] 각각을 화면에서 없앤다.

답이 다를 시 (else)
quizView[num].classList.add("dislike") : quizView[num] 각각에 dislike 클래스를 적용
quizConfirm[num].style.display = "none" : quizConfirm[num] 각각을 화면에서 없앤다.
quizResult[num].style.display = "block" : quizResult[num] 각각을 화면에 다시 나타나게 해준다.
quizInput[num].style.display = "none" : quizInput[num] 각각을 화면에서 없앤다.
