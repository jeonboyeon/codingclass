배열 메서드 | join( ), push( ), pop( )

이번 글은 배열 메서드의 join( ), push( ), pop( ) 에 대한 포스터 입니다.


배열과 메서드

배열은 다양한 메서드를 제공하는데 오늘은 다양한 메서드 중
요소 추가 | 제거 메서드에 대해서 알아볼 것입니다.

배열 메서드 | join( )

join( ) 메서드는 배열의 모든 요소를 연결하여 하나의 문자열로 만든 후, 숫자로 반환한다. 


const arrNum = [100, 200, 300, 400, 500];

const text1 = arrNum.join('');
const text2 = arrNum.join(' ');
const text3 = arrNum.join('★');
const text4 = arrNum.join('-');

위 const arrNum = [100, 200, 300, 400, 500]; 에
text1, text2, text3, text4에 join()메서드를 넣어 출력시키게 되면

arrNum.join('') = 100200300400500
arrNum.join(' ') = 100 200 300 400 500
arrNum.join('★') = 100★200★300★400★500
arrNum.join('-') = 100-200-300-400-500
라는 답이 출력되게 된다.

위에 설명한대로 join()메서드는 배열 모든 요소를 연결하여 하나의 문자열로 만들기 때문에
join()괄호 안에 넣은 것들이 배열 사이사이에 들어가 하나의 문자열로 되는 것 이다.

배열 메서드 | push( )

push( ) 메서드는 배열 끝에 하나 이상의 요소를 추가 시키고, 배열의 새로운 길이를 반환한다. 

const arrNum = [100, 200, 300, 400, 500];
    
const arrPush = arrNum.push(600);

위 const arrNum = [100, 200, 300, 400, 500]; 에
arrPush = arrNum.push(600); 으로 arrPush 를 출력하게 되면 어떻게 될까?

위에 설명을 생각하며 답을 도출하면 100 200 300 400 500 '600' 마지막에 600이 추가되어 답이 나올 것 같지만,
arrPush 값의 답은 6이 나오게된다.
즉 push() 메서드를 쓴 상수값을 출력 시키게 되면 arrNum 에 600까지 추가시켜 배열의 총 숫자를 구하게 되고,
100 200 300 400 500 600 을 출력하고 싶다면 arrNum 을 써서 출력을 시켜줘야 하는 것 이다.

배열 메서드 | pop( )

pop( ) 메서드는 배열 마지막 요소를 제거하고, 그 요소를 반환한다. 

const arrNum = [100, 200, 300, 400, 500];
    
const arrPush = arrNum.pop();

위 설명 처럼 pop( ) 메서드는 배열 마지막 요소를 제거 후, 그 요소를 반환해준다.

그 설명을 이해하고 위 arrPush 의 값을 출력하게 되면
arrNum 배열의 맨 마지막 요소인 500을 제거 후, 500을 반환하여 닶은 500이 출력되게 된다.
그렇게 arrNum 의 값도 같이 출력을 하게 된다면 500이 빠진 100 200 300 400 만 출력이 되는 것이다.


이렇게 오늘은
배열의 모든 요소를 연결하여 하나의 문자열로 만든 후, 숫자로 반환 하는 join( ) 메서드
배열 끝에 하나 이상의 요소를 추가 시키고, 배열의 새로운 길이를 반환 하는 push( ) 메서드
배열 마지막 요소를 제거하고, 그 요소를 반환 하는 pop( ) 메서드에 대해서 공부해 보았습니다.
