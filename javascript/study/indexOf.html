JAVASCRIPT indexOf( ) 메서드

안녕하세요 👋🏻 이번 글은 JacaScript indexOf( ) 메서드 에 대해 알아보는 글입니다.

indexOf( )가 무엇인가요?

indexOf( )란 특정 문자가 문자열의 어느 index에 위치하는지 찾는 방법의 메서드입니다.
indexOf( )는 문자열 특정 문자의 위치를 찾은 후 그 문자열을 숫자로 반환합니다.


◾️ indexOf( ) 사용법

📍 indexOf( ) 메서드의 사용법은
     string(문자열).indexOf(searchvalue, position)
     즉 문자열.indexOf(찾고싶은 문자열, 찾고싶은 문자열을 찾기 시작 할 위치) 순으로 적어야합니다.


✔️ 사용법 예시

const str1 = "javascript reference";

const currentStr1 = str1.indexOf("javascript"); //0
💡 찾는 문자열이 없을 때는 숫자 -1을 반환하며, 문자열을 찾을 때 대소문자를 구별합니다.

◾️ 예제 이해하기 👩🏻‍💻

     📍 오늘 배운 indexOf( )의 예제를 적어보겠습니다. 차근차근 눈여겨 보고, 이해 해봅시다!


✔️ indexOf( ) 예제

const str1 = "javascript reference";

const currentStr1 = str1.indexOf("javascript"); // 0
const currentStr2 = str1.indexOf("reference"); // 11
const currentStr3 = str1.indexOf("j"); // 0
const currentStr4 = str1.indexOf("a"); // 1 중복 될 때는 가장 처음에 있는것을 서치한다.
const currentStr5 = str1.indexOf("v"); // 2
const currentStr6 = str1.indexOf("jquery"); // -1 데이터가 없을 때는 -1 이 나온다.
const currentStr7 = str1.indexOf("b"); // -1 데이터가 없을 때는 -1 이 나온다.
💡 첫 번째 const currentStr1 = str1.indexOf("javascript"); 의 답은 0입니다.
     완전히 이해가 된 건 아니지만 javascript는 0부터 시작이 되기 때문에 0 으로 반환이 된 것 같네요.

     두 번째 const currentStr2 = str1.indexOf("reference"); 의 답은 11입니다.
     reference 는 r이 11번 째에 있으므로 반환된 숫자 값은 11로 나오는 것 같습니다.

     이렇게 각 단어의 위치를 숫자로 변환하여 출력 시키고, 위 예제에서 보았 듯이 jquery 와 b 는
     str1에 저장되어 있지 않은 문자열이므로 숫자 -1을 반환하게 된겁니다.

✔️ 아래는 예시를 좀 더 추가 해놓은 것입니다. (주석은 답입니다.)

const str1 = "javascript reference";
    
const currentStr8 = str1.indexOf("javascript", 0); // 0
const currentStr9 = str1.indexOf("javascript", 1); // -1
const currentStr10 = str1.indexOf("reference", 0); // 11
const currentStr11 = str1.indexOf("reference", 1); // 11
const currentStr12 = str1.indexOf("reference", 11); // 11
const currentStr13 = str1.indexOf("reference", 12); // -1
