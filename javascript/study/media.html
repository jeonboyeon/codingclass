CSS 미디어 쿼리 (media Query) 알아보기

안녕하세요 😎 이번 글은 CSS 미디어 쿼리 (media Query)에 대해 알아보는 글입니다.

CSS 미디어 쿼리 (media Query) 란?

미디어 쿼리는 CSS에서 어떤 스타일을 선택적으로 적용하고 싶을 때 사용한다.
뷰포트의 해상도에 따라 CSS를 분기시키는 미디어쿼리는 반응형 프로젝트에서 반드시 필요한 기법입니다.

◾️ <link ~ media="">

<link rel="stylesheet" media="all and (min-width: 1000px)" href="style_pc.css">

위 HTML 코드는 최소 가로폭 1000px, 즉 1000px 보다 넓은 해상도일 때 style_pc.css를 연결한다는 뜻이다.

태블릿이나 모바일 기기마다 가로폭이 다른데, 크롬 브라우저에서 웹 페이지에 우클릭 > 검사 를 클릭하고
왼쪽 맨 위 화살표 아이콘 옆에 있는 휴대폰, 태블릿 아이콘을 누르게 되면
각 기기의 해상도도 확인이 가능하고, 기기의 모델도 추가할 수 있다.


◽️ 모바일 기기 해상도 표

◾️ @media

첫 로딩 때 성능이 저하되지 않도록 CSS 파일을 하나로 만들어서
CSS 내부에서 조건에 따라 분기시키는 것이 일반적인 형태의 반응형웹 CSS 이다. 

@media all and (min-width: 1000px) {
        /* 모든 기기에서 해상도가 최소 1000px 인 겨우 적용할 요소들을 적어주면 된다. */
    }

위 @meida 옆에 all 은 Media Type 을 나타낸다.
삽입 위치는 and 앞, 뒤에 띄어 쓰면 된다.


◽️ 미디어 타입 종류

Media Type 설명
all 모든 미디어 장치에 사용 (기본값)
print 프린터에 사용
screen PC, 태블릿, 스마트폰에 사용
speech 스크린리더기가 페이지를 읽는데 사용


✔️ 예시 01 번

body {
        color: red;    
    }

    @media only screen and (max-width:768px) {
        body {
            color: blue;
        }
    }

위 예시는 원래 글자 색이 빨간색이지만
화면이 768px 보다 작아졌을 때 글자를 파란색으로 변경하겠다는 뜻 입니다.

📍 작은 장치에서 페이지를 더 빠르게 로딩시키고 싶다면 모바일 우선으로 설계하면 됩니다.


✔️ 예시 02 번

body {
        color: blue;    
    }

    @media only screen and (max-width:768px) {
        body {
            color: red;
        }
    }

위 2번 예시는 1번과 반대로 원래 글자 색이 파란색이지만
화면이 768px 보다 작아졌을 때 글자를 빨간색으로 변경하겠다는 뜻 입니다.


✔️ 예시 03 번

미디어 쿼리를 사용하여 브라우저의 방향에 따라 다른 레이아웃을 설정 할 수 있다.

@media only screen and (oreientation: portrait) {
        body {
            background: skyblue;
        }
    }

위 예시는 모바일 기기가 세로 방향이 되면, 배경색을 하늘색으로 변경하겠다는 뜻 입니다.


✔️ 예시 04 번

@media only screen and (oreientation: landscape) {
        body {
            background: lightgreen;
        }
    }

위 예시는 모바일 기기가 가로 방향이 되면, 배경색을 연두색으로 변경하겠다는 뜻 입니다.


✔️ 예시 05 번

쉼표를 사용하여 조건을 추가 할 수 있다.

@media only screen and (max-width: 900px) and (min-width: 600px), (min-width: 1000px) {
        body {
            color: red;
        }
    }

위 예시는 가로가 600px 에서 900 사이일 때,
또 1000px 이상일 때 글자색을 빨간색으로 변경한다는 뜻이다.

◾️ 미디어 쿼리 (media Query) 실습 👩🏻‍💻
