자바스크립트 연산자

프로그램에 필요한 수식을 만들 때 필요한 것이 연산자입니다.
자바스크립트 연산자에는 산술연산자, 대입연사자, 비교연산자, 논리연산자 등이 있습니다.

산술 연산자

사칙연산을 다루는 기본적이면서도 가장 많이 사용되는 연산자

종류 설명
+ 더하기 연산
문자와 문자, 문자와 변수 연결
- 빼기 연산
* 곱하기 연산
/ 몫을 구하는 연산
% 나머지를 구하는 연산
++ 1씩 증가 시키는 연산
-- 1씩 감소 시키는 연산

대입 연산자

변수에 대입하는 연산자

종류 설명
= num = 1 : 1은 우변의 1값을 좌변 num 변수에 대입
+= num += 2 는 num = num + 2 와 같다.
- num -= 2 는 num = num - 2 와 같다.
* num *= 2 는 num = num * 2 와 같다.
/ num /= 2 는 num = num / 2 와 같다.
% num %= 2 는 num = num % 2 와 같다.

비교 연산자

값을 비교하여 참과 거짓으로 반환하는 연산자

종류 설명
> a > b 가 참이면 참, 거짓이면 거짓
< a > b 가 참이면 참, 거짓이면 거짓
>= a >= b 가 참이면 참, 거짓이면 거짓
<= a <= b 가 참이면 참, 거짓이면 거짓
== a == b ㅣ a 와 b 가 같으면 참, 같지 않으면 거짓
ex) 1 == '1' [참]
!== a != b ㅣ a 와 b 가 같지 않으면 참, 같으면 거짓
=== a === b ㅣ a 와 b 가 같고 데이터 타입도 같으면 참, 같으면 거짓
ex) 1 == '1' [거짓]
!== a !== b ㅣ a 와 b 가 같지 않거나 데이터 타입이 같지 않으면 참, 같으면 거짓

논리 연산자

연산식의 조합을 참인지 거짓인지 판단하는 연산자

종류 설명
&& (AND연산자) a && b ㅣ a 와 b 모두 참 일 때 참이 된다.
|| (OR연산자) a || b ㅣ a 와 b 중 하나만 참이면 참이 된다.
! (NOT연산자) ! a ㅣ a 가 참이면 거짓, 거짓이면 참으로 바꿔준다.
