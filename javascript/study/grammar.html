CSS 기본 문법 알아보기

안녕하세요 😘 이번 글은 CSS 기본 문법에 대해 알아보는 글입니다.

CSS 기본 문법에는 어떤 것들이 있나요?

CSS 기본 문법에는 내부 스타일시트, 외부 스타일시트, 인라인 스타일시트 등이 있습니다.
실제로는 외부 스타일시트를 많이 사용하는 편이지만, 간단한 스타일 속성을 줄 때는 내부 스타일시트를 사용하기도 합니다.

간단한 스타일을 줄 때는 내부 스타일시트 !

내부 스타일시트는 CSS 를 HTML 문서 내 <head></head> 태그 안에 <style></style> 태그를 넣어
HTML과 CSS 가 함께 저장 되도록 하는 형태입니다. 


<!DOCUMENT html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>내부 스타일시트</title>
    <style type="text/css">
        p {
            color: red;   /* p 태그의 글씨 색을 빨간색으로 바꿔준다. */
        }
    </style>
<head>
<body>
    <p>이 문장의 색은 빨강입니다.</p>
</body>
</html>

■ 추가 설명
<style type="text/css">의 type="text/css" 는 콘텐츠가 CSS 임을 나타내주는데,
HTML5 부터 <style> 태그의 type 속성 기본값이 "text/css" 로 설정되어 있기 때문에 따로 줄 필요가 사라졌다.

애용하자 ! 외부 스타일시트 !

외부 스타일시트는 스타일 속성을 HTML 과 별도의 CSS 파일을 만들어
HTML 문서에<link> 태그로 연결시켜 실행하는 형태입니다. 


<!DOCUMENT html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>내부 스타일시트</title>
    <link rel="stylesheet" href="style.css"> <!-- href="style.css" | CSS 파일 경로 -->
<head>
<body>
    <p>이 문장의 색은 빨강입니다.</p>
</body>
</html>

/* CSS 외부 파일 | style.css */

@charset "UTF-8";
p {
    color: red;
}

다른 CSS 파일을 불러오고 싶다면? @import !

자신의 CSS 파일에 다른 CSS 를 가져올 때 쓰인다.
@import 는 @charset 규칙을 제외하고 다른 규칙들보다 선행하여 CSS에 삽입하여야 한다. (거의 맨 윗줄에 기술을 합니다.) 


<!DOCUMENT html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>내부 스타일시트</title>
    <style>
        @import url("style.css");
    </style>
<head>
<body>
    <p>이 문장의 색은 빨강입니다.</p>
</body>
</html>

또한 @import는 어떤 CSS 파일 안에 공통 CSS를 불러들일 경우에도 사용할 수 있다.

/* CSS 외부 파일 | style.css */

@charset "UTF-8";
@import url("common.css"); /* style.css 외부 CSS */

@import는 공통으로 들어가야 할 스타일을 따로 저장하여 다른 CSS에 삽입시켜 사용할 수 있도록 해준다.
하지만 CSS 파일을 너무 여러 개로 나누게 되면,
오히려 복잡해져서 유지보수가 힘들게 되니 용량이 적다면 하나의 파일로 저장하는 것이 좋다.

마지막 인라인 스타일!

인라인 스타일은 HTML 태그에 직접 CSS 속성을 주는 기법으로, 해당 태그에 style="" 형식으로 기술한다.
내부 스타일이나 외부 스타일에서 기술된 속성보다 가장 우선으로 적용된다.
하지만 스타일을 공통으로 수정할 수 없고 각 HTML 파일을 열어 각각 편집해야 하므로 일반적으로 많이 사용하지 않는 방법이다. 


<!DOCUMENT html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>내부 스타일시트</title>
<head>
<body>
    <p style="color: red;">이 문장의 색은 빨강입니다.</p>
</body>
</html>

공부를 마치며 📋

오늘은 CSS 기본 문법들에 대해서 알아보았습니다.
HTML 내부에 작성하는 내부 스타일시트,
태그에 직접 적용시키는 인라인 스타일,
HTML 외부에 CSS 파일을 만들어 HTML 에 link 로 연결시키는 외부 스타일
CSS 파일 내에 다른 CSS 파일을 넣을 수 있는 @import 방법까지 총 4개의 기본 문법에 대해 알아보았습니다.
