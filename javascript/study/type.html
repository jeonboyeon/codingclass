데이터 타입

변수에 저장되는 데이터 유형을 일컫는 말

데이터 타입의 종류는 무엇이 있나요?

데이터 타입의 유형으론 원시(Primitive)타입, 객체(Object)타입으로 나뉩니다.
원시 타입에는 number, string, boolean, undefined, null, symbol 등이 있고,
객체 타입에는 function, object, array 등이 있습니다.

객체 타입 유형 알아보기

첫번 째 : number

정수, 소수점, 지수를 표현할 수 있다.

var num1 = 10;
var num2 = 10.5;
var num3 = le+2;

console.log(numl); // 10
console.log(num2); // 10.5
console.log(num3); // 100 le+2 는 1*10의 2승을 의미

두번 째 : string

문자를 표현할 때 쓰는 데이터
'' 혹은 "" 으로 표현한다.

var str1 = ' 문자 ' ;
var str2 = " 문자 " ;

문자 안에 문자가 들어갈 경우
var str1 = '문자 "문자"';
var str1 = "문자 "문자'";

세번 째 : blooean

참과 거짓으로 값을 표현하는 데이터

var temp1 = (5 > 4);
var temp2 = (5 < 4);

console.log(temp1); // 결과 값 : 참
console.log(temp2); // 결과 값 : 거짓

네번 째 : undefined

변수를 선언하고 데이터 값은 지정하지 않았을 때 결과 값으로 나타나는 데이터

var temp1 = 10;
var temp2;

console.log(temp1); // 결과 값 : 10
console.log(temp2); // 결과 값 : undefined

다섯번 째 : null

변수를 빈 상태로 초기화 하거나, 데이터는 저장했지만 값이 존재하지 않을 때 null값으로 반환 한다.

var obj = 10;
obj = null;
console.log(obj); // 결과 값 : null | 변수를 빈 상태로 만든다.

var obj = document.getElementById('gnb');
console.log(obj); // 결과 값 : 변수에 데이터 값을 저장했으나, 값이 존재하지않아 null을 저장
