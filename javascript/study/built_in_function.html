JAVASCRIPT 내장함수의 종류

안녕하세요 😉 이번 글은 JacaScript 내장함수 종류에 대해 알아보는 글입니다.

내장함수가 무엇인가요?

내장함수란 JavaScript 내에서 기본적으로 제공해준는 함수를 의미합니다.


[ 내장 함수의 종류 ]

◾️ 인코딩 & 디코딩 함수

📍 URL 주소 끝에 쿼리 정보를 전송하여 데이터를 처리해야 되는 프로그램의 경우
     한글과 같은 유니코드 문자가 포함되어 있으면 오류가 발생합니다.
     이런 경우를 대비해 인코딩 함수를 이용하여 문자를 부호화시키고,
     부호화된 문자를 다시 디코딩 함수를 이용하여 원래 문자로 되돌릴 수 있다.


◽️ 인코딩 & 디코딩 설명 표

함수명 설명
encodeURIComponent( ) 영문, 숫자와 ( ) - . ~ * ! ' 을 제외한 문자를 인코딩한다.
decodeURIComponent( ) 인코딩(encodeURIComponent( )) 의 디코딩 함수이다.

✔️ 인코딩 & 디코딩 예시

// 인코딩 & 디코딩 함수
var encodeStr = "자바스크립트";
console.log(encodeURIComponent(encodeStr));

var decodeStr = encodeURIComponent(encodeStr);
console.log(decodeURIComponent(dncodeStr));

✔️ 예시 출력 값 (console 확인)

◾️ 숫자 유 | 무한 값 판단 함수

📍 숫자 유 | 무한 값 판단 함수는 숫자가 맞는지 아닌지, 숫자가 유한값인지 무한값인지 판단하는 함수이다.


◽️ 숫자 유 | 무한 값 판단 함수 설명 표

함수명 설명
isNaN( ) 숫자인지 아닌지 판변하는 함수이다.
숫자이면 false를 반환하고, 숫자가 아니면 true를 반환한다.

💡 NaN은 Not a Number의 약자이다.

isFinite( ) 유한값인지 무한값인지 판변하는 함수이다.
유한값이면 true를 반환, 무한값이면 false를 반환한다.

✔️ 유 | 무한 값 판별 함수 예시

// 유 | 무한 값 판별 함수 
var num1 = "숫자";

if (!isNaN(num1)) {
    console.log("숫자입니다.");
} else {
    console.log("숫자가 아닙니다.");
};

var num2 = 1 / 0;

if (!isFinite(num2)) {
    console.log("유한값");
} else {
    console.log("무한값");
};

✔️ 예시 출력 값 (console 확인)

◾️ 숫자 | 문자 변환 함수

📍 문자를 숫자로 변환하거나, 숫자를 문자로 변환해주는 함수이다.


◽️ 숫자 | 문자 변환 함수 설명 표

함수명 설명
Number( ) 문자를 숫자로 변환한다.
parselnt( ) 숫자와 문자가 포함되어 있는 경우
정수 부분만 숫자로 변환한다.
parseFloat( ) 숫자를 문자로 변환한다.
String( ) 숫자인지 아닌지 판변하는 함수이다.
숫자이면 false를 반환하고, 숫자가 아니면 true를 반환한다.

✔️ 숫자 | 문자 변환 함수 예시

// 숫자 | 문자 변환 함수
var num1 = "10";

console.log(Number(num1));

var num2 = "100px";

console.log(parseInt(num2));

var num3 = "33.3%";

console.log(parseFloat(num3));

var num4 = 10;

console.log(typeof num4);
console.log(typeof String(num4));

✔️ 예시 출력 값 (console 확인)

◾️ 자바스크립트 코드 변경 함수

📍 문자를 자바스크립트 코드로 변경한다.


◽️ 자바스크립트 코드 변경 함수 설명 표

함수명 설명
eval( ) 문자를 자바스크립트 코드로 변경한다.

✔️ 자바스크립트 코드 변경 함수 예시

// 자바스크립트 코드 변경 함수
var str1 = "var num1 = 10";
var str2 = "var num2 = 20";

eval(str1);
eval(str2);

console.log(num1 + num2);

✔️ 예시 출력 값 (console 확인)
