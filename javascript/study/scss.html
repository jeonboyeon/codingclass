CSS | SCSS Variables (변수)

SCSS Variables (변수) 사용법

SCSS란?

Syntactically Awesome StyleSheet의 약어이며 코드의 재활용성을 올리고,
가독성을 올리는 등 CSS에서 보이던 단점을 보완하고, 개발의 효율을 올리기 위해 등장한 CSS 전처리기 언어이다.


◾️ SCSS Variables (변수) 사용법

📍 변수를 스타일시트 전체에서 재사용하려는 정보를 저장하는 방법으로 생각하시면 됩니다.
    색상, 글꼴 스택 또는 재사용하고 싶은 CSS 값과 같은 항목을 저장할 수 있습니다.
    Sass는 $ 기호를 사용하여 무언가를 변수로 만듭니다. 다음은 예입니다.


✔️ 사용법 예시

$font-stack: Helvetica, sans-serif; /* 변수 font-stack에 폰트 Helvetica, sans-serif 저장 */
    $primary-color: #333; /* 컬러 #333을 저장 */
    
    body {
      font: 100% $font-stack; /* 저장한 변수를 가져와 재사용 */
      color: $primary-color;
    }

✔️ 위 SCSS를 CSS로 compiled 시켜서 확인

body {
    font: 100% Helvetica, sans-serif;
    color: #333;
  }
